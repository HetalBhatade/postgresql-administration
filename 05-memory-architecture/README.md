
# PostgreSQL Memory Architecture

## Objective

This section documents how PostgreSQL uses memory to improve database performance and reduce disk I/O.

It explains the two primary memory areas used by PostgreSQL, their key components, and how memory settings influence query execution.

The content is based on hands-on practice together with the PostgreSQL Official Documentation.
