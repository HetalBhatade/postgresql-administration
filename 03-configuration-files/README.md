# PostgreSQL Configuration Files

## Objective

This section documents the PostgreSQL configuration files that control PostgreSQL server behavior and client authentication.

It explains the purpose of each configuration file, where it is located, how PostgreSQL loads configuration settings, and how configuration changes are applied.

The content is based on hands-on practice together with the PostgreSQL Official Documentation.

## Topics Covered

- PostgreSQL configuration file overview
- Configuration loading order
- postgresql.conf
- postgresql.auto.conf
- pg_hba.conf
- pg_ident.conf
- Viewing configuration file locations
- Viewing configuration parameters
- ALTER SYSTEM configuration changes
- Reloading configuration
- Configuration file verification
