# PostgreSQL Database Lab Setup

## Objective

This section documents the creation of a PostgreSQL sample database used throughout the PostgreSQL Administration lab.

The database contains schemas, tables, indexes, views, functions, triggers, and sample data that are used for backup, recovery, replication, maintenance, and performance testing.

The content is based on hands-on practice using PostgreSQL.

## Topics Covered

- Create Database
- Create Schema
- Create Tables
- Create Indexes
- Create View
- Create Function
- Create Trigger
- Insert Sample Data
- Verify Database Objects
