# PostgreSQL Background Processes

## Objective

This section documents the PostgreSQL background processes responsible for maintaining database performance, durability, recovery, logging, replication, and routine maintenance.

It explains the purpose of the major background processes, how to view them from the operating system, and demonstrates PostgreSQL's process monitoring and automatic process recovery.

The content is based on hands-on practice together with the PostgreSQL Official Documentation.
